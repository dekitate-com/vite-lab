## Vite + Tanstack Router の最小構成サンプルリポジトリへようこそ！

### 技術構成

このプロジェクトは以下の技術スタックで構成されています：

- **Vite** - ビルドツール
- **vitejs/plugin-react** - plugin (今後 Rolldown対応が先にこちらに入るという予測のもとswcではなく通常盤を使用)
- **React** - UI ライブラリ
- **TanStack Router** - ルーティングライブラリ（自動コード分割対応）

### 主な機能
- ファイルベースルーティング（TanStack Router Plugin）
- 自動コード分割（`autoCodeSplitting: true`）
- React 19 対応
